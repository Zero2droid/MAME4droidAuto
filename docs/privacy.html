  3. 3. 为应对突发公共卫生事件，或者紧急情况下为保护自然人的生命健康和财产安全所必需；
  4. 4. 为公共利益实施新闻报道、舆论监督等行为，在合理的范围内处理个人信息；
  5. 5. 依照本法规定在合理的范围内处理个人自行公开或者其他已经合法公开的个人信息；
